Version Display Component Test

Note: The version display component should appear as a semi-transparent panel in the bottom-right corner of the screen. It will attempt to fetch version information via gRPC-web, but will fall back to mock data if the server is not available.

This page demonstrates the <version-display> web component integration. The component should:

  • Appear in the bottom-right corner
  • Have a semi-transparent dark background
  • Display version, commit, and build time information
  • Show loading states during fetch operations
  • Gracefully handle errors with fallback data
