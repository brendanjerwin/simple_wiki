Version Display Component Test

Note: The version display component should appear as a small, transparent horizontal bar in the bottom-right corner of the screen. It will attempt to fetch version information via gRPC-web, but will remain blank if the server is not available.

This page demonstrates the <version-display> web component integration. The component should:

  • Appear as a small horizontal bar in the bottom-right corner with minimal margin
  • Have a very transparent background that darkens on hover
  • Display version, commit, and build time information in a single row
  • Show loading indicators during fetch operations
  • Remain blank if the server is not available (no fallback data)
